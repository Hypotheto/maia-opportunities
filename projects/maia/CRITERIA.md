# Maia Ranking Criteria

## Primary fit factors
Rank opportunities using judgment across these factors:

1. Field relevance
- creative direction
- exhibitions
- adjacent creative / design / cultural fields

2. Environment quality
- strong working environment
- real project exposure
- credible organisation or team
- people Maia could learn from

3. Presence / access
- on-site is preferred
- hybrid is acceptable if unusually strong
- remote is lower priority

4. Growth value
- learning potential
- network expansion
- portfolio / experience value
- contribution to design thinking / creative direction skills

5. Duration fit
- ideal: 1-3 months
- acceptable if strategically strong

6. Geography fit
- Europe easiest
- outside Europe acceptable if the opportunity is strong enough

7. Compensation
- paid preferred
- unpaid acceptable only if exposure / learning / network upside is exceptional

## Suggested labels
- High-interest: strong alignment across several important factors
- Worth reviewing: plausible fit with some uncertainty or weaker tradeoffs
- Low-priority: possible but weak fit versus Maia's stated goals
- Hold / unclear: not enough information yet

## Important rule
A low score is not automatic rejection. The system is meant to preserve optionality, not eliminate options too early.
