# Maia Site

Open this file in a browser:

- `index.html`

## What this site is

A lightweight static review board for Maia's opportunities.
It is meant to let Maia scan quickly, then click deeper only where something deserves attention.

## Current structure

- `index.html` — landing page / review board
- `styles.css` — shared styling
- `assets/` — screenshots and image assets
- `opportunities/` — per-opportunity report pages
