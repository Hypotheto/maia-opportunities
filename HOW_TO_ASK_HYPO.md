# HOW TO ASK HYPO

A short cheat sheet for getting the right kind of help fast.

## 1. Memory

### Save something
- **remember this** → save to daily memory
- **remember this long-term** → save to long-term memory
- **save this to daily memory**
- **save this as a dated memory**

### Retrieve something
- **what do you remember about X?**
- **when did we talk about X?**
- **when did I tell you about X?**
- **what is saved about X?**
- **check durable memory for X**

## 2. Thinking / clarity
- **help me think this through**
- **structure this for me**
- **what is the real issue here?**
- **what am I missing?**
- **what matters most?**
- **give me the sharp version**

## 3. Strategy
- **what is the best move here?**
- **what are my options?**
- **how do I position this?**
- **what should I say to X?**
- **how do I approach this person?**
- **what is the risk here?**

## 4. Writing
- **draft this**
- **write me the message**
- **make it shorter**
- **make it stronger**
- **make it sound like me**
- **make it sound like you**
- **make it less formal**
- **make it more convincing**

## 5. Work / organization
- **turn this into tasks**
- **summarize decisions**
- **what are the next steps?**
- **who owns what?**
- **what should we do first?**
- **make me a plan for this week**
- **track this project with me**

## 6. Research
- **research this**
- **look into X**
- **monitor this**
- **what should we be watching?**
- **who are the key people here?**
- **what are the important signals?**

## 7. Best-practice patterns

### If you want memory
Say it explicitly.
- **remember this**
- **remember this long-term**

### If you want a date
Ask for the date directly.
- **when did we discuss X?**
- **what date is this saved under?**

### If you want a recommendation
Ask for judgment, not just information.
- **what do you think I should do?**
- **what is your read?**

### If you want me to challenge you
- **push back on this**
- **play devil’s advocate**
- **tell me what’s weak here**

### If you want speed
Start with the output you want.
- **draft a message to AM**
- **summarize this in 5 lines**
- **give me 3 options**

## 8. The simplest house rules
- **remember this** = save to daily memory
- **remember this long-term** = save to permanent memory
- **when did we talk about X?** = retrieve date if supported by evidence
- **help me think** = structure + judgment
- **draft this** = output first, discussion second

## 9. Good examples
- **Remember this long-term: my strongest lane at Tethys is likely special projects close to AM.**
- **When did we first discuss Seraf?**
- **Help me think through Monday’s meeting with AM.**
- **Draft a short message to CK.**
- **Turn this into tasks and next steps.**
- **Push back on my plan.**

---

If I’m not giving you what you want, the fastest correction is:
- **No, I want X**
- **Be more practical**
- **Shorter**
- **Go deeper**
- **Stop explaining and just do it**
